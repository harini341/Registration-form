# Registration Form with Real-Time Validation

## Project Description

This project is a simple **Registration Form** built using **HTML, CSS, and JavaScript**.
It performs **real-time form validation** to ensure that users enter valid information before submitting the form.

The form contains three fields:

* Name
* Email
* Password

The **Submit button remains disabled** until all fields are correctly validated.

## Features

* Real-time input validation using JavaScript
* Error messages displayed only for incorrect inputs
* Name field cannot be empty
* Email must follow a valid email format
* Password must contain **at least 6 characters**
* Submit button is enabled only when all inputs are valid

## Technologies Used

* HTML
* CSS
* JavaScript

## How It Works

1. The user enters details in the form fields.
2. JavaScript checks each input in real-time.
3. If the input is invalid, an error message is displayed.
4. When all fields are valid, the **Submit button becomes active**.

## Usage

1. Download or clone the repository.
2. Open the `index.html` file in your browser.
3. Enter valid details in all fields to enable the submit button.

## Learning Outcome

This project helps in understanding:

* Form validation in JavaScript
* Event handling
* Basic front-end development concepts
* Improving user input handling in web forms

## Conclusion

This project demonstrates how to implement a simple registration form with real-time validation using JavaScript. By validating user inputs such as name, email, and password, the form ensures that only correct information is submitted. It also improves user experience by displaying error messages only when necessary and enabling the submit button only after all fields are valid. This project helps in understanding the basics of client-side form validation and interactive web development.
